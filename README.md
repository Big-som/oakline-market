Oakline Market Infrastructure Deployment on AWS

Introduction

Oakline Market is a growing grocery business seeking to expand its online presence. The objective of this project was to design and deploy a secure, scalable, and highly available AWS cloud infrastructure to support the company's website, product image storage, staff access management, and future business growth.

Project Objectives
Implement secure user access management.
Create cloud storage for product images.
Deploy a Linux web server.
Configure persistent storage.
Implement load balancing.
Enable automatic scaling for high availability.
